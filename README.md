# Bank-Customer-Segmentation
Power BI project analyzing customer segmentation, transaction behavior, and profitability insights
🔷 Project Overview

This project focuses on analyzing bank customer data to uncover meaningful insights related to customer demographics, transaction behavior, segmentation, and profitability.

Using Power BI, interactive dashboards were developed to transform raw data into clear and actionable insights. The project demonstrates how data visualization can support better decision-making in the banking domain.

🎯 Objectives
Analyze customer data to identify patterns and trends
Segment customers based on behavior and financial activity
Evaluate profitability and potential risk indicators
Present insights through interactive dashboards
📌 Dashboards
📍 1. Customer Demographics Dashboard
Visualizes customer distribution by age, gender, region, and occupation
Helps understand the overall customer profile
Supports targeted marketing strategies
📍 2. Transaction Behaviour Insights
Analyzes transaction frequency, volume, and spending patterns
Identifies active and inactive customers
Helps improve customer engagement
📍 3. Customer Segmentation Analysis
Groups customers into different segments based on behavior
Identifies high-value and low-engagement customers
Enables targeted decision-making
📍 4. Profitability and Risk Indicators
Evaluates customer contribution to revenue
Identifies potential financial risk patterns
Supports balanced business strategies
🛠️ Tools & Technologies
Power BI
Data Visualization
Data Analysis
📈 Key Insights
Customer distribution varies across different demographic groups
A small group of customers contributes significantly to overall revenue
Transaction patterns help identify engagement levels
Risk analysis is essential for maintaining financial stability
